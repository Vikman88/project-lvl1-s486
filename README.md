## Setup

```sh
$ git clone https://github.com/Vikman88/project-lvl1-s486.git
$ make install
```

## Run brain-even

```sh
$ make brain-even
```

## Run brain-calc

```sh
$ make brain-calc
```

## Run brain-gcd

```sh
$ make brain-gcd
```
